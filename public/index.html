Brotli/gzip precompression, immutable caching for hashed assets, ETag revalidation, and HTTP range requests.

🔒 Safe by default

Path-traversal and dotfile protection, security headers, and method restrictions — zero dependencies.

🚀 Deploy free

A committed GitHub Actions workflow publishes to GitHub Pages on every push. Netlify, Vercel & Docker configs included.

🛠 Great DX

npm run dev gives you live-reload. npm run preview builds and serves the exact production output.

Three commands

  1. npm run dev — edit public/ with live-reload
  2. npm run build — produce optimized dist/
  3. git push — auto-deploy to your free host
